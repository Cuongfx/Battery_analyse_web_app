# README

This dataset consists of 240 502030-size Li-polymer batteries whose positive electrode is lithium nickel manganese cobalt oxide and negative electrode is graphite. The electrolyte is unknown. The nominal capacity of these batteries is 0.25 Ah. There are 202 different charge/discharge protocols in this dataset, with 30 degrees Celsius as the working temperature.



If you use this dataset, please cite the original papers:

- **Predicting Battery Lifetime Under Varying Usage Conditions from Early Aging Data** *2024 Cell Reports Physical Science*
