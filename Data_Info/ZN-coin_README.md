# README

This dataset consists of 95 commercial Li-ion batteries in a coin format whose positive electrode is commercial manganese dioxide cathodes(MnO2), the negative electrode is zinc metal, and the electrolyte is disclosed. The discharge capacity of the 10<sup>th</sup> cycle was taken as the nominal capacity for each battery. All batteries underwent the same charge/discharge protocol at 25 degrees Celsius.



If you use this dataset, please cite the original papers:

- **BatteryLife: A Comprehensive Dataset and Benchmark for Battery Life Prediction**

## Details for processed data 

- **Package:** 1---CR2032, 2---CR2025, 3---CR2016
- **Current:** 1---A current density of 400 mA/m<sup>2</sup>
- **Anode:** 1---99.99% pure zinc foil
- **Cathode:** 1---commercial manganese dioxide cathodes
- **Electrolyte:** For the protection of the intellectual property rights of the data producers, we do not disclose the composition of the 63 different electrolytes used in this dataset, but use numbers 1-63 to indicate the electrolytes used
- **Temperature:** 1---25℃, 2---30℃, 3---40℃
