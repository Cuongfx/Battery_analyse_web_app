# README

Combined LFP dataset — cells with a lithium-iron-phosphate (LiFePO4) positive electrode and a graphite negative electrode, aggregated from two sources: MATR (169 cells, 18650, 1.1 Ah) and HUST (77 cells, 18650, 1.1 Ah). All 246 cells are cycled at an ambient working temperature of 30 degrees Celsius, with varied fast-charging protocols.
