# README

This dataset consists of 23 commercial Li-ion batteries in an 18650 cylindrical format whose positive electrode is LiNi0.5Co0.2Mn0.3O2 and negative electrode is graphite. The electrolyte is unknown. The nominal capacity of these batteries is 2 Ah. This dataset includes 2 different operation conditions with a working temperature of 20 degrees Celsius.



If you use this dataset, please cite the original papers:

- **Physics-informed neural network for lithium-ion battery degradation stable modeling and prognosis** *2024 Nature Communications*
