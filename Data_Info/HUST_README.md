# README

This dataset consists of 77 commercial Li-ion batteries in an 18650 cylindrical format whose positive electrode is lithium-iron-phosphate, and the negative electrode is graphite. The electrolyte is unknown.  The nominal capacity is 1.1 Ah. All batteries are cycled with the same protocols with a working temperature of 30 degrees Celsius.



If you use this dataset, please cite the original papers:

- **Real-time personalized health status prediction of lithium-ion batteries using deep transfer learning** *2022 Energy Environ. Sci.*
