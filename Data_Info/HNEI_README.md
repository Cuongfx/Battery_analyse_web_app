# README

This dataset consists of 14 commercial Li-ion batteries in an 18650 format whose positive electrode is LiCoO2 and LiNi0.4Co0.4Mn0.2O2, the negative electrode is graphite, and the electrolyte is unknown. The nominal capacity of all batteries is 2.8 Ah. All batteries were cycled under a working temperature of 25 degrees Celsius with varied fast-charging protocols and identical discharge protocols. 2 different fast-charging protocols are included in this dataset.



If you use this dataset, please cite the original papers:

- **Intrinsic Variability in the Degradation of a Batch of Commercial 18650 Lithium-Ion Cells** *2018 Energies*
