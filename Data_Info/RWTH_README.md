# README

This dataset consists of 48 commercial Li-ion batteries in an 18650 cylindrical format whose positive electrode is lithium nickel manganese cobalt oxide (NMC), while the negative electrode is carbon, and the electrolyte is unknown.  The nominal capacity is 3 Ah. All batteries are cycled with the same protocols with a working temperature of 25 degrees Celsius.



If you use this dataset, please cite the original papers:

- **One-shot battery degradation trajectory prediction with deep learning** *2021 Journal of Power Sources*
