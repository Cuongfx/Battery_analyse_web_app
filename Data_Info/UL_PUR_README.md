# README

This dataset consists of 10 commercial Li-ion batteries in an 18650 cylindrical format whose positive electrode is LiCoO2 and LiNi0.4Co0.4Mn0.2O2, the negative electrode is graphite, and the electrolyte is unknown. The nominal capacity of all batteries is 3.4 Ah. All batteries were cycled under a working temperature of 23 degrees Celsius with varied fast-charging protocols and identical discharge protocols. All batteries underwent the same cycling protocol.



If you use this dataset, please cite the original papers:

- **Degradation-Safety Analytics in Lithium-Ion Cells: Part I. Aging under Charge/Discharge Cycling** *2020 J. Electrochem. Soc.*
