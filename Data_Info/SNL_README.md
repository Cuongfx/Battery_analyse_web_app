# README

This dataset consists of 52 commercial Li-ion batteries in an 18650 cylindrical format whose positive electrode covers LiFePO4, LiNi0.81Co0.14Al0.05O2, and LiNi0.84Mn0.06Co0.1O2, while the negative electrode is graphite, and the electrolyte is unknown. There are 3 nominal capacities consisting of 1.1 Ah, 3.2 Ah, and 3.0 Ah. There are 23 different protocols in this dataset with 3 different working temperatures (15°C, 25°C and 35°C).



If you use this dataset, please cite the original papers:

- **Degradation of Commercial Lithium-Ion Cells as a Function of Chemistry and Cycling Conditions** *2020 J. Electrochem. Soc.*
