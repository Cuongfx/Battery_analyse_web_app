# README

This dataset consists of 86 commercial Li-ion batteries in an 18650 cylindrical format whose positive electrode is NMC and negative electrode is graphite. The electrolyte is unknown. The nominal capacity of these batteries is 2.4 Ah. All batteries underwent different formation protocols but were cycled with16 different charge/discharge protocols under a working temperature of 25 degrees Celsius.

If you use this dataset, please cite the original papers:

- **Deep sorting of reused batteries for enabling long term consistency grouping with unknown prior conditions** *2025 Cell Reports Physical Science*
