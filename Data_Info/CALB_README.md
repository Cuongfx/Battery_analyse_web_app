# README

This dataset consists of 27 commercial Li-ion batteries in a prismatic format whose positive electrode is NMC, the negative electrode is graphite, and the electrolyte is not disclosed. The nominal capacity of all batteries is 58 Ah. There are 2 different protocols with 4 different working temperatures (0°C,25°C,35°Cand45°C). Other details are shown in the [Battery-Life/BatteryLife_Raw at main](https://huggingface.co/datasets/Battery-Life/BatteryLife_Raw/tree/main/CALB).

**Note: **All batteries cycling under 0°C were underwent an accelerated aging test.

**Acknowledgement:** Thanks to CALB Group Co., Ltd. for providing the raw data.
