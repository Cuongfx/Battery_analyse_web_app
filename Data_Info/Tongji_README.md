# README

This dataset consists of 108 commercial Li-ion batteries in an 18650 cylindrical format covering 3 chemical systems consisting of 3 different positive electrodes(Li0.86Ni0.86Co0.11Al0.03 O2/Li0.84(Ni0.83Co0.11Mn 0.07)O2/42 wt.% Li(NiCoMn)O2 blended with 58 wt.% Li(NiCoAl)O2), 2 different negative electrodes(Graphite/Graphite+2 wt.% Si), and an identical electrolyte. This dataset includes 2 nominal capacities (3.5 Ah and 2.5 Ah), 3 operating temperatures, and 6 charge/discharge protocols.



If you use this dataset, please cite the original papers:

- **Data-driven capacity estimation of commercial lithium-ion batteries from voltage relaxation** *2022 Nature Communications.*
